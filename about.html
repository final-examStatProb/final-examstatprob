Measures of Central Tendency (Ungrouped Data)

Mean

Definition: The mean is the sum of all data points divided by the number of data points.

Formula: Mean = (Î£x) / n

Example: If the data points are 5, 7, and 9, the mean is (5 + 7 + 9) / 3 = 7.

Median

Definition: The median is the middle value when the data points are arranged in ascending order.

Steps to Find:

    Arrange the data in order.

    If there is an odd number of data points, the median is the middle value.

    If there is an even number of data points, the median is the average of the two middle values.

Example: Data set: 3, 7, 9. The median is 7.

Mode

Definition: The mode is the data point that appears most frequently in a set.

How to Identify: Look for the value that repeats the most.

Example: In the data set 1, 2, 2, 3, 3, 3, the mode is 3.

Fundamental Counting Principle

Explanation: This principle states that if one event can occur in m ways and a second event can occur in n ways, then the two events can occur in m Ã— n ways.

Example: Choosing an outfit with 3 shirts and 2 pants can be done in 3 Ã— 2 = 6 ways.

Permutations

Definition: A permutation is an arrangement of objects in a specific order.

Formula: P(n, r) = n! / (n - r)!

Example: Arranging 3 objects from 5: P(5, 3) = 5! / (5 - 3)! = 60.

Practice Problems

1. Calculate the Mean:

Data: 10, 15, 20, 25, 30. Find the mean.

2. Find the Median:

Data: 2, 4, 6, 8, 10. Find the median.

3. Find the Mode:

Data: 1, 2, 2, 3, 4, 4, 4. Find the mode.

4. Apply the Fundamental Counting Principle:

There are 4 shirts and 3 pants. How many different outfits can be formed?

5. Calculate Permutations:

How many ways can you arrange 4 books from a shelf of 6 books?

6. Calculate Combinations:

How many ways can you choose 3 players from a team of 7?
